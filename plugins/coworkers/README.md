# Coworkers

A Claude Code skill containing virtual coworker personas — distilled from real colleagues' design docs, code reviews, emails, and decisions. So they can keep working with us.

## Coworkers

| Persona | Role | Style |
|---------|------|-------|
| Jack | Embedded architect | Embedded architecture, safety-first, naming precision, platform unification |

## Install

Claude Code marketplace:

```bash
/plugin marketplace add yuhua0731/my-marketplace
/plugin install coworkers@my-marketplace
```

Codex marketplace:

```bash
View coworkers from the my-marketplace plugin list.
```

# OmniFlow Common Terms

## Product Names

- 慧仓穿云箭: OmniFlow
- product_line: `omniflow`

## Generic Terms

- goods-to-person: goods or totes are delivered to an operator or station
- tote-to-person: tote-level movement to a workstation
- storage rack: high-density rack body
- workstation / WS: operator picking workstation when the project uses this abbreviation
- scheduler: software layer assigning tasks, reservations, routes, and robot actions
- robot fleet: multiple robot types or roles cooperating in one storage system

## C134 Terms That May Recur

- Ant: ground mobile robot in C134; verify naming in other projects
- Mantis: rack/aisle handling robot in C134; verify naming in other projects
- WLED: C134 workstation light-strip equipment; not automatically an Ant component
- PT/PD: C134 transfer or handoff positions; verify definitions per project

## Guardrails

- Treat project-specific abbreviations as unknown until defined in that project.
- Do not assume C134 `WS`, `WLED`, `PT`, `PD`, Ant, or Mantis semantics apply unchanged to another OmniFlow site.

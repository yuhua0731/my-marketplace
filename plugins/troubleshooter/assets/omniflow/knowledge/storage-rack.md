# OmniFlow Storage Rack Knowledge

## Use For

- rack-side storage/retrieval failures
- tote stuck, missing, mispositioned, or collision with rack/transfer point
- abnormal behavior tied to rack geometry or storage location

## Common Fault Branches

- tote not at expected position
- rack/track/transfer-point geometry mismatch
- project configuration differs from physical layout
- robot offset or access-node parameters mismatch the site
- mechanical interference, deformation, obstruction, or contamination

## Evidence To Request

- exact rack/aisle/bay/layer/tote coordinates
- video or image of tote and robot pose
- project layout/config file when available
- robot handling logs and scheduler command timeline
